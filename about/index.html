About

I’m a network engineer, specialising in Cisco networking technologies and with a ﬁrst-class honours networking degree.

I currently work full-time in Leeds for one of the world’s biggest bookmakers. Using the Agile methodology, I am part of a specialist “SDN” team that collaborates on network automation projects.

I created this space as a place to document my personal development; from a traditional network engineer using the CLI to manually configure one device at a time, to a “NetDevOps” engineer utilising tools such as Ansible, Git and Python programming. This shift to a “network as code” thought process will enable me to collaborate on, test and deploy fully automated configuration changes to the entire network.

One of my future goals is to develop my orchestration skills and integrate the automation code I develop into a CI/CD pipeline.
